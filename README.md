# muddyRainSass

——由 Muddyrain 开发的样式包

如何引入样式包：

```js
// scss文件 引入方式
import "muddy-rain-sass/dist/app.scss";
// css 文件引入
import "muddy-rain-sass/dist/app.css";
// css压缩文件
import "muddy-rain-sass/dist/app.css.map";
```

目录介绍 dist :

```js
align.scss —— 对齐方式样式
bg.scss -- 背景样式
box.scss -- 盒模型样式
common.scss -- 公共样式 -》 .no-number 隐藏 数字输入框小按钮
display.scss -- 展示样式
flex.scss -- 弹性盒子样式
float.scss -- 浮动样式
mixin.scss -- 嵌入语法样式
position.scss -- 定位样式
reset.scss -- 重置样式 -》 Eric Meyer's Reset CSS v2.0 (http://meyerweb.com/eric/tools/css/reset/)  http://cssreset.com
shadow.scss -- 阴影样式
sizing.scss -- 尺寸样式
text.scss -- 文本样式
transition.scss -- 过渡样式 -》 可以加载过渡组件
variables.scss -- 变量样式
border.scss -- 边框样式
button.scss -- 按钮样式
overflow.scss -- 溢出样式
table.scss -- 边框样式
input.scss -- 输入框样式
image.scss -- 图片样式
```
